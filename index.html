Qué es Sublime Text 2?

...y por qué debería utilizarlo


Creado por Jon Skinner como una extension VIM

Multiplataforma: OSX, Linux & Windows


¡Al fin un editor nos pone a todos de acuerdo!

Ligero (sobre 7mb) y portable

Interfaz limpia, intuitiva y configurable

Rápido, muy rápido


Escrito en C++ y Python

Soporte nativo a infinidad de lenguajes

H

Python, C++, XML, PHP, JavaScript, HTML, CSS...

MULTI: layout, select & cursor

Configuración "al gusto" de KeyBindings


Es mi teclado y me lo f*@k# como quiero =)

Soporte de Snippets y Plugins

Gran comunidad de desarrolladores

y mucho más: VIM mode, minimap, ...

Preferences

...ST2 a tu gusto

Dentro de la carpeta mágica Packages...


  • plugins (.py)
  • settings (.sublime-settings)
  • key maps (.sublime-keymap)
  • build systems (.sublime-build)
  • menus (.sublime-menu)
  • snippets (.sublime-snippet)
  • macros (.sublime-macro)
  • sublime themes (.sublime-theme)
  • syntax definitions (.tmLanguage)
  • syntax themes (.tmTheme)
  • preferences (.tmPreferences)

Settings (.sublime-settings)


Configuración del comportamiento del editor


Existen dos grupos principales: default & User


Siempre, siempre, guarda en Settings User

KeyBindings (.sublime-keymap)


Configuración de tooooodos los atajos de teclado


Siempre, siempre, guarda en KeyBindings User
